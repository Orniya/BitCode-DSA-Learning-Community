# Bon Appétit 🍽️

🟡 **Easy**

## 📌 Overview

A billing fairness problem where we check if Anna was overcharged for shared items.

## 🏆 Objective

- Given bill array, excluded item, and charged amount
- Determine if split was fair

## 📂 Problem Breakdown

- Remove item Anna didn’t eat
- Split remaining cost equally
- Compare expected vs actual payment

## 🎯 Key Takeaways

- Array manipulation
- Fair division logic
- Error checking in calculations
