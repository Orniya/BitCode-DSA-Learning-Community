#  Sock Merchant

🟢 **Easy**

## 📌 Overview

This is a beginner-friendly warm-up problem focused on counting pairs within a collection of items. It helps you practice array traversal and basic counting techniques commonly used in DSA.

---

## 🏆 Objective

- Given an array of integers representing items (e.g., socks), identify matching pairs.
- Count how many pairs can be formed from the array.
- Return the total number of pairs.

---

## 📂 Problem Breakdown

- You are given a list of integers.
- Each integer represents a type of item.
- Your task is to:
  - Count how many times each value appears
  - Form pairs from matching values
  - Return the total number of pairs

---

## 📌 Example Scenario

If given:
```
arr = [1, 2, 1, 2, 1, 3, 2]
```

Pairs are:
- 1 → 1 pair
- 2 → 1 pair

So the output is:
```
2
```
