# Cats and a Mouse 🐱🐭

🟡 **Easy**

## 📌 Overview

A logic-based comparison problem where two cats chase a mouse on a number line.

## 🏆 Objective

- Given positions of Cat A, Cat B, and Mouse C
- Determine which cat reaches the mouse first

## 📂 Problem Breakdown

- Compute distances:
  - |A - C|
  - |B - C|
- Compare distances and decide outcome

## 🎯 Key Takeaways

- Absolute difference calculations
- Decision-making logic
- Basic simulation reasoning
