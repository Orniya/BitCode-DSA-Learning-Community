# 🟢 HackerRank Warm-up (Fresh Set – 10 Questions)

A collection of beginner-friendly HackerRank problems designed to strengthen **fundamental programming skills in JavaScript**.  
These problems focus on **arrays, loops, conditionals, matrices, and basic string manipulation**.

---

## 🏆 Objective

- Build strong programming fundamentals using JavaScript
- Improve logical thinking through small problem-solving tasks
- Practice common coding patterns used in interviews
- Gain confidence in handling arrays, strings, and math-based logic

---

## 📂 Problems List

---

### 1️⃣ Solve Me First
A basic function-based problem where you return the sum of two integers.  
Focuses on understanding input/output and function structure.

🔗 https://www.hackerrank.com/challenges/solve-me-first/problem

---

### 2️⃣ Simple Array Sum
Compute the sum of all elements in an integer array using iteration.  
Strengthens array traversal and accumulation logic.

🔗 https://www.hackerrank.com/challenges/simple-array-sum/problem

---

### 3️⃣ Compare the Triplets
Compare two arrays element by element and assign scores based on comparisons.  
Builds conditional logic and array indexing skills.

🔗 https://www.hackerrank.com/challenges/compare-the-triplets/problem

---

### 4️⃣ Diagonal Difference
Find the absolute difference between primary and secondary diagonal sums in a square matrix.  
Improves understanding of 2D arrays and matrix traversal.

🔗 https://www.hackerrank.com/challenges/diagonal-difference/problem

---

### 5️⃣ Plus Minus
Calculate ratios of positive, negative, and zero values in an array.  
Focuses on counting frequency and computing proportions.

🔗 https://www.hackerrank.com/challenges/plus-minus/problem

---

### 6️⃣ Mini-Max Sum
Find minimum and maximum sums of four elements out of five integers.  
Develops sorting and selective summation logic.

🔗 https://www.hackerrank.com/challenges/mini-max-sum/problem

---

### 7️⃣ Sock Merchant
Count the number of matching pairs in an array of socks.  
Introduces frequency counting and pairing logic.

🔗 https://www.hackerrank.com/challenges/sock-merchant/problem

---

### 8️⃣ Time Conversion
Convert time from 12-hour AM/PM format to 24-hour format.  
Strengthens string manipulation and time formatting logic.

🔗 https://www.hackerrank.com/challenges/time-conversion/problem

---

### 9️⃣ Staircase
Print a right-aligned staircase pattern using symbols.  
Practices nested loops and pattern building.

🔗 https://www.hackerrank.com/challenges/staircase/problem

---

### 🔟 A Very Big Sum
Compute the sum of very large integers that may exceed standard integer limits.  
Focuses on handling large number arithmetic safely.

🔗 https://www.hackerrank.com/challenges/a-very-big-sum/problem

---

## 🎯 Key Takeaways

- Strong foundation in **loops, arrays, and conditionals**
- Introduction to **matrix and string problems**
- Improved **pattern recognition and logic building**
- Essential preparation for **DSA and coding interviews**

---

⭐ “Small problems, solved consistently, build strong engineers.”
