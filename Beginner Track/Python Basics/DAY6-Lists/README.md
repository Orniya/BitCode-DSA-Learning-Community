# 🐍 Python Basics - Day 7: Lists (Arrays)

## 📖 Overview

In this challenge, you'll learn about Python lists, which are the equivalent of arrays in other programming languages. Lists allow you to store, access, and manipulate multiple values in a single structure.

## 🎯 Objectives

* Understand how to create and use lists in Python.
* Access elements using indexing.
* Modify lists by adding and removing elements.
* Use loops to iterate through list elements.

## 📂 Problem Breakdown

Lists are one of the most important data structures in Python:

* They store multiple values in an ordered format.
* Each element can be accessed using its index (starting from 0).
* Lists are mutable, meaning they can be changed after creation.

Common operations include:

* Adding elements (`append`, `insert`)
* Removing elements (`pop`, `remove`)
* Traversing using loops

## 📌 Example Scenario

Given a list of numbers, perform operations such as updating values, removing elements, and iterating through the list to produce a result.
