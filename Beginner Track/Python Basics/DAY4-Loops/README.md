# 🐍 Python Basics - Day 5: Loops

## 📖 Overview

In this challenge, you'll learn how to use loops to automate repetitive tasks in Python. Loops are one of the most important building blocks in programming and are heavily used in problem-solving and DSA.

## 🎯 Objectives

* Understand how loops work in Python.
* Use `for` and `while` loops effectively.
* Iterate through sequences like lists and strings.
* Perform repeated operations efficiently.

## 📂 Problem Breakdown

Loops allow you to execute a block of code multiple times:

* `for` loops are used when the number of iterations is known.
* `while` loops run until a condition becomes false.
* Looping is commonly used to process arrays, strings, and ranges.

## 📌 Example Scenario

Given a list of numbers, use a loop to process each element and perform required operations such as summing or filtering values.
