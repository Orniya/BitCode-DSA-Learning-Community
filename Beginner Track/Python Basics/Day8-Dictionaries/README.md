# 🐍 Python Basics - Day 10: Dictionaries

## 📖 Overview

In this challenge, you'll learn about dictionaries in Python. Dictionaries store data in key-value pairs and are one of the most important structures in DSA.

## 🎯 Objectives

* Understand key-value pair structure.
* Add, update, and access dictionary elements.
* Use dictionaries for fast lookups.

## 📂 Problem Breakdown

Dictionaries are powerful mappings:

* Each key is unique.
* Values can be accessed using keys.
* They are highly efficient for searching data.

## 📌 Example Scenario

Given a set of data, store and retrieve values using dictionary keys for quick access.
