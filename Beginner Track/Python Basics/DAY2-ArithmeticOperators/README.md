# 🐍 Python Basics - Day 3: Arithmetic Operators

## 📖 Overview

In this challenge, you'll learn how to perform mathematical operations using Python's arithmetic operators. These operators are used extensively in programming, from simple calculations to complex algorithmic problems.

## 🎯 Objectives

* Work with Python's basic arithmetic operators.
* Perform calculations using numerical inputs.
* Understand how expressions are evaluated.
* Display computed results correctly.

## 📂 Problem Breakdown

Python provides several arithmetic operators that allow us to manipulate numbers efficiently:

* Addition (`+`)
* Subtraction (`-`)
* Multiplication (`*`)
* Division (`/`)
* Modulus (`%`)

You will apply these operators to given values and produce the expected output.

## 📌 Example Scenario

Given two numbers, calculate and display their sum, difference, product, and quotient using Python's arithmetic operators.
