# 🐍 Python Basics - Day 8: Tuples

## 📖 Overview

In this challenge, you'll explore tuples in Python. Tuples are similar to lists but are immutable, making them useful for storing fixed collections of data.

## 🎯 Objectives

* Understand what tuples are and how they differ from lists.
* Create and access tuple elements.
* Learn when to use tuples in programming.

## 📂 Problem Breakdown

Tuples are ordered collections:

* They cannot be changed after creation.
* They are useful for storing fixed data.
* They support indexing and iteration.

## 📌 Example Scenario

Given a tuple of values, perform operations such as accessing elements or using them in calculations.

## 🎯 Key Takeaways

* Understanding immutability in Python.
* Knowing when to use tuples instead of lists.
