# 🐍 Python Basics - Day 9: Sets

## 📖 Overview

In this challenge, you'll learn about sets in Python. Sets are unordered collections that store unique elements and are widely used for eliminating duplicates and fast membership checks.

## 🎯 Objectives

* Understand what sets are.
* Learn how to add and remove elements.
* Perform membership testing.
* Use sets for removing duplicates.

## 📂 Problem Breakdown

Sets are powerful data structures:

* They store only unique values.
* They do not maintain order.
* They allow fast lookup operations.

## 📌 Example Scenario

Given a list of values, convert it into a set to remove duplicates and perform operations like union or intersection.
