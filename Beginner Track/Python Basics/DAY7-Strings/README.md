In this challenge, you'll learn how to work with strings in Python. Strings are one of the most commonly used data types in programming and appear in almost every DSA problem.

## 🎯 Objectives

* Perform basic string operations.

## 📂 Problem Breakdown

Strings in Python are sequences of characters:

* You can access characters using indexes.
* Strings support slicing to extract parts of text.
* They can be iterated using loops.
