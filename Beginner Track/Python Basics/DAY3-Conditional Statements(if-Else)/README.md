# 🐍 Python Basics - Day 4: Conditional Statements (If-Else)

## 📖 Overview

In this challenge, you'll learn how to control the flow of a program using conditional statements. Conditions allow your code to make decisions and execute different actions depending on the given input.

## 🎯 Objectives

* Understand how conditional statements work in Python.
* Use `if`, `elif`, and `else` to create decision-making logic.
* Evaluate expressions using comparison operators.
* Write programs that respond differently based on input values.

## 📂 Problem Breakdown

Conditional statements help programs choose between multiple possible outcomes:

* `if` executes code when a condition is true.
* `elif` checks additional conditions when previous ones are false.
* `else` provides a default action when no conditions match.

These structures are fundamental to solving programming and DSA problems.

## 📌 Example Scenario

Given a number, determine its category based on a set of predefined conditions and display the appropriate result.
