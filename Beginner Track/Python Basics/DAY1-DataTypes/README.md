# 🐍 Python Basics - Day 2: Data Types

## 📖 Overview

In this challenge, you'll explore Python's fundamental data types and learn how to work with different kinds of values. Understanding data types is essential, as they form the foundation of every program and algorithm you'll write.

## 🎯 Objectives

* Understand common Python data types.
* Work with integers, floating-point numbers, strings, and booleans.
* Perform basic operations using different types of data.
* Learn how Python handles input and output.

## 📂 Problem Breakdown

* Variables can store different kinds of data.
* Different data types support different operations.
* Sometimes values need to be converted from one type to another before they can be used correctly.

## 📌 Example Scenario

Given an integer, a floating-point number, and a string, perform the required operations and display the expected output.
