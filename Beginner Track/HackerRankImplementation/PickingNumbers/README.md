# Picking Numbers 📊

🟡 **Easy**

## 📌 Overview

This is an array-based optimization problem. The goal is to find the largest subset of integers where the absolute difference between any two elements is at most 1.

## 🏆 Objective

- Given an array of integers
- Find the largest subset where:
  - |a[i] - a[j]| ≤ 1
- Return the maximum size of such a subset

## 📂 Problem Breakdown

- Count frequency of each number
- Compare adjacent values (x and x+1)
- Find maximum sum of two neighboring frequencies

## 🎯 Key Takeaways

- Frequency counting technique
- Pattern recognition in arrays
- Optimization using constraints
