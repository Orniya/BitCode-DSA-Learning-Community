# Drawing Book 📖

🟡 **Easy**

## 📌 Overview

A mathematical problem involving minimum page flips to reach a target page in a book.

## 🏆 Objective

- Given total pages and target page
- Find minimum flips from front or back

## 📂 Problem Breakdown

- Each flip turns 2 pages
- Compute flips from both directions
- Return minimum value

## 🎯 Key Takeaways

- Counting patterns
- Bidirectional optimization
- Integer division logic
