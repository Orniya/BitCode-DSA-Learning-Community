# Easy Implementation Challenges (EASY)


## 📌 Overview

This collection of HackerRank **implementation problems** focuses on strengthening core **JavaScript problem-solving skills**.  
Each problem is designed to improve logical thinking using **arrays, loops, conditionals, arithmetic operations, and date handling** through real-world styled scenarios.

---

## 🏆 Objective

- Practice **easy-level implementation problems** using Python.
- Build strong understanding of **logic building and structured thinking**.
- Improve confidence in handling **array manipulation, math-based logic, and edge cases**.
- Prepare for **DSA foundations and coding interviews**.

---

## 📂 Problems Covered


### 1️⃣ Picking Numbers 📊
Find the largest subset of integers where the absolute difference between any two elements is less than or equal to 1.

### 2️⃣ Cats and a Mouse 🐱🐭
Compare distances of two cats from a mouse on a number line and determine which cat reaches it first.

### 3️⃣ Drawing Book 📖
Calculate the minimum number of page flips required to reach a given page from either the front or back of a book.

### 4️⃣ Bon Appétit 🍽️
Check whether Anna was correctly charged for a shared meal by validating the split bill calculation.

### 5️⃣6️ Day of the Programmer 📅
Determine the exact date of the 256th day of the year, considering leap years and calendar transition rules.
