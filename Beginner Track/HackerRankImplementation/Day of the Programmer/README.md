Day of the Programmer 📅

🟡 **Easy**

## 📌 Overview

A calendar logic problem where we find the 256th day of a given year considering different calendar systems.

## 🏆 Objective

- Given a year
- Find the 256th day of that year
- Handle calendar transitions and leap years

## 📂 Problem Breakdown

- Determine calendar system (Julian/Gregorian/transition year)
- Adjust February days for leap years
- Calculate exact date

## 🎯 Key Takeaways

- Calendar-based logic
- Edge case handling
- Historical system differences
